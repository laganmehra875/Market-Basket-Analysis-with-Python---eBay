# Market-Basket-Analysis-with-Python-eBay Machine learning 

**🎯 Project Objective**

The objective of this project is to analyze customer purchasing behavior using eBay transaction and survey data in order to:

 1. Understand customer demographics and buying patterns
 2. Identify frequently purchased product categories
 3. Analyze cart abandonment behavior
 4. Evaluate the effectiveness of personalized recommendations
 5. Segment customers based on behavioral patterns using clustering
 6. Provide actionable business insights to improve customer engagement and revenue
 
This analysis aims to support data-driven decision-making for marketing, product recommendations, and customer retention strategies.
